# 自动bugku签到

打开bugku主页自动签到，已经测试，但是不能完全脱离浏览器，必须每天谷歌浏览器启动才可以运行

# 使用方式：

谷歌浏览器 -> 右上角扩展程序 -> 管理扩展程序 -> 右上角开发者模式 -> 左上角加载已解压的扩展程序 -> 把旁边那个"bugku-zdqian" 文件夹选上； 然后保证插件打开即可(这个插件默认叫 Auto Check-in Plugin 1.0)
